# react-native-yandex-mapkit

React Native implementation of Yandex Mapkit

## Installation

```sh
npm install react-native-yandex-mapkit
```

## Usage

```js
import { YandexMapkitView } from "react-native-yandex-mapkit";

// ...

<YandexMapkitView color="tomato" />
```
